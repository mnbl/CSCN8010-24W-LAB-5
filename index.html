Performing cross-validation for Polynomial Models

In[16]:
poly_results = []
for name, model in poly_models.items():
    poly_cv_results = cross_validate(model, X_train, Y_train, scoring=scoring, cv=5)
    r2_mean = poly_cv_results['test_r2'].mean()
    r2_std = poly_cv_results['test_r2'].std()
    mae_mean = -poly_cv_results['test_mae'].mean()
    mae_std = poly_cv_results['test_mae'].std()
    transformed_X = PolynomialFeatures(degree=model.steps[0][1].degree).fit_transform(X_train)
    mape_mean = np.mean(np.abs((Y_train - model.fit(transformed_X, Y_train).predict(transformed_X)) / Y_train)) * 100
    poly_results.append([name, r2_mean, r2_std, mae_mean, mae_std, mape_mean])

# Train a degree 2 polynomial regression model
poly_features = PolynomialFeatures(degree=2)
X_poly = poly_features.fit_transform(X_train)  # Transform training data
model = LinearRegression().fit(X_poly, Y_train)  # Train the model

# Predict on the test data
X_test_poly = poly_features.transform(X_test)
Y_pred = model.predict(X_test_poly)

# Calculate R-Squared
r_squared = r2_score(Y_test, Y_pred)

# Calculate MAPE
mape = np.mean(np.abs((Y_test - Y_pred) / Y_test)) * 100

# Calculate MAE
mae = mean_absolute_error(Y_test, Y_pred)

print(f"Performance on Test Data:\nR2 Score: {r_squared}\nMean Absolute Error: {mape}\nMean Absolute Percentage Error: {mae}")

Conclusion

The polynomial regression model with a degree of 2 performed moderately well on the test data, explaining about 46.70% of the variance with an R-squared score and having an average error of around 40.60. However, its average percentage error was high at approximately 44.32%. Despite its performance, the model has limitations: it is sensitive to outliers and noise, and depend heavily on the quality of input features. To improve, considering more complex models, adding relevant features, and refining outlier detection and feature engineering are recommended.
